API Documentation

All Quotes

To request all quotes, please make a GET request to:

https://parks-and-rec-quotes.herokuapp.com/a/

Your data will come back to you in the following format:

{"Code":200,"Quotes":{"Speaker":"Quote", "Speaker2":"Quote2"}}

Random Quote

To request one random quote, please make a GET request to:

https://parks-and-rec-quotes.herokuapp.com/r/

Your data will come back to you in the following format:

{"Code":200, "Quote":"Quote Text", "Speaker":"Speaker Text"}
